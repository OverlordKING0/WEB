Ноаостной сайт это полнофункциональная веб-платформа для создания, публикации и потребления новостей. Проект объединяет возможности современных СМИ и удобного блога, где каждый зарегистрированный пользователь может стать автором, а читатель — найти только релевантный контент.

## Описание
Новостной портал созданный с использованием микрофреймворка Flask в рамках обучения 

## Стек технологий
* Python
* Flask
* WTForms
* SQLAlchemy

1. Склонируйте репозиторий
2. Создайте и активируйте виртуальное окружение .venv
3. ````commandline
   py -m venv .vene
   sours .venv/Scripts/activate
3. Установите зависимости. Они лежат в файле requirements.txt
4. Создайте файл .env и укажите настройки подключения к DB и другим параметрам
5. Запустите приложение через терминал
